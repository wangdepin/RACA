# Public Projects

Code for your publicly released projects — open-source tools, published paper code, shared datasets, etc.

RACA can read and edit code here, but **will never push to a remote repository without your explicit approval**. Public-facing code needs your sign-off before it goes out.

```
public_projects/
  my-paper-code/       ← git repo (public GitHub)
  shared-benchmark/    ← git repo (public GitHub)
```

When you're ready to push, RACA will show you the diff and ask for confirmation.
